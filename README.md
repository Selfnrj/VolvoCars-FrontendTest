## Getting Started

First, install packages:

```bash
npm i
```

Second, run the development server:

```bash
npm run dev
```

Third, run the production build:

```bash
npm run build
```

```bash
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
